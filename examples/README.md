# Examples

Three worked examples demonstrating `regime-shap` across domains:

- `01_uk_gdp.ipynb` — Macroeconomic forecasting (the original dissertation use case)
- `02_stock_volatility.ipynb` — Financial application (regime detection in market volatility)
- `03_energy_demand.ipynb` — Energy application (electricity demand across COVID lockdown)

## Running

Install with the examples extras:

```bash
pip install regime-shap[examples]
```

Launch Jupyter:

```bash
jupyter notebook
```

Each notebook is self-contained and uses publicly available data.
